Functions

The following functions are available globally.

  • bind(_:_:)

    Function to bind a genaric type and closure together. This is used to allow us to respond to geometry event.

    
 func layoutTraits(`for` proxy: GeometryProxy) -> LayoutTraits {
    ...
    return traits
 }

 GeometryReader { proxy in
          bind( layoutTraits(for: proxy) ) { layoutTraits in
            // You now have access to the traits that are bound to the proxy.
          }
 }

    Declaration

    Swift

    func bind<T, U>(_ x: T, _ closure: (T) -> U) -> U

    Parameters

    x

    Generic Type that will pass a result to the closure
    closure

    The closure that will take the output of the input function that is provided and perform operations on it.
  • >*(_:)

    Shorthand used to easily convert a View to AnyView.

    Parameters

    lhs
